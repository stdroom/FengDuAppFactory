    Youmi Android SDK OfferWall Developer Guide¶

    Tip

    Welcome to download Youmi Ad SDK. In order to ensure you receiving Youmi ad normally and guarantee you ad revenue, we may need you to stick to the steps listed in the documentation under the doc file, or please take referrance to the demo source code.

    Please read the following tips before integration:

    1. When you are creating the app on the Youmi backend to get APPID and KEY, please be noted that you MUST select the right OS;
    2. One pair of APPID and KEY only can be applied to one app; P.S: please do NOT use the same APPID and KEY in different apps, otherwise you may NOT able to receive ad normally;
    3. You can receive test ad after integration; After test and confirmation, please upload the apk/ipa on Youmi website for review; After review approval you will receive official ads that brings you revenue immediately;
    4. Please noticed that Youmi ad do NOT support Simulator, You MUST test on real mobile Device; Besides, Android pad is not appliable here for test, please test Android ad on phone only.

    1. Youmi Android SDK general configuration¶

    Before add the OfferWall, please make sure you have already finished the general configuration according to Youmi Android SDK General Configuration Developer Guide .

    2. Configure OfferWall Components¶

    Please copy the below configuration code and paste it into file AndroidManifest.xml :

    <activity
    android:name="net.youmi.android.AdBrowser"
    android:configChanges="keyboard|keyboardHidden|orientation|screenSize"
    android:theme="@android:style/Theme.Light.NoTitleBar" >
</activity>
<service
    android:name="net.youmi.android.AdService"
    android:exported="false" >
</service>
<receiver
    android:name="net.youmi.android.AdReceiver" >
    <intent-filter>
        <action android:name="android.intent.action.PACKAGE_ADDED" />
        <data android:scheme="package" />
    </intent-filter>
</receiver>
<service
    android:name="net.youmi.android.ExpService"
    android:exported="false" >
</service>

    Danger

    Be Noticed:

    • ExpService is a configuration item and must be added. Otherwise, the currency points can’t be acquired normally.

    3. Offerwall code integration (Important)¶

    1) Initialization and related operation¶

    Please start the initialization before implementing the function of Offerwall (e.g. invoke the following code in onCreate of the first Activity)

    import net.youmi.android.offers.OffersManager;
...
OffersManager.getInstance(Context context).onAppLaunch();

    In the exit of the app (e.g. in the onDestroy method of Activity), invoke following code to complete the Offerwall resource recycle.

    import net.youmi.android.offers.OffersManager;
...
OffersManager.getInstance(Context context).onAppExit();

    2) Display the full-screen OfferWall¶

    Please invoke the following code to display the full-screen Offerwall:

    import net.youmi.android.offers.OffersManager;
...
OffersManager.getInstance(Context context).showOffersWall();

// Meanwhile this method is also applied to the following reload
// Support full-screen Offerwall exit monitoring callback
OffersManager.getInstance(Context context).showOffersWall(Interface_ActivityListener listener);

    Interface_ActivityListener defines as follow:：

    public interface Interface_ActivityListener {

    /**
     * full-screen sharewall Activity send callback when invoking onDestory,
     * and it will executed in the UI thread
     */
    public void onActivityDestroy(Context context);
}

    3) Display the floating half-screen Sharewall¶

    Please invoke the following code in UI thread, to display the floating half-screen Sharewall:

    import net.youmi.android.offers.OffersManager.
...
OffersManager.getInstance(Context context).showOffersWallDialog(Activity activity);

// Meanwhile this method supports following reload

// input dialog to end the monitor
// OffersManager.getInstance(Context context).showOffersWallDialog(Activity activity, OffersWallDialogListener listener);

// input dialog Wide, height and pixel
// OffersManager.getInstance(Context context).showOffersWallDialog(Activity activity, int width, int height);

// input dialog wide, height and pixel, and end the monitor
// OffersManager.getInstance(Context context).showOffersWallDialog(Activity activity, int width, int height, OffersWallDialogListener listener);

// input the dialog wide and height proportion (0~1)
// OffersManager.getInstance(Context context).showOffersWallDialog(Activity activity, double scaleOfScreenWidth, double scaleOfScreenHeight);

// input the dialog wide and height proportion (0~1) and end the monitor
// OffersManager.getInstance(Context context).showOffersWallDialog(Activity activity, double scaleOfScreenWidth, double scaleOfScreenHeight, OffersWallDialogListener listener);

    OffersWallDialogListener definition as below:

    public interface OffersWallDialogListener {

    /**
     * send callback when end the dialog
     */
    public void onDialogClose();
}
