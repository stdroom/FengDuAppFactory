    Youmi Android SDK General Configuration Developer Guide¶

    Tip

    Welcome to download Youmi Ad SDK; In order to ensure you receiving Youmi ad normally and guarantee you ad revenue, we may need you to stick to the steps listed in the documentation under the doc file, or please take referrance to the demo source code.

    Please read the following tips before integration:

    1. When you are creating the app on the Youmi backend to get APPID and KEY, please be noted that you MUST select the right OS;
    2. One pair of APPID and KEY only can be applied to one app; P.S: please do NOT use the same APPID and KEY in different apps, otherwise you may NOT able to receive ad normally;
    3. You can receive test ad after integration; After test and confirmation, please upload the apk/ipa on Youmi website for review; After review approval you will receive official ads that brings you revenue immediately;
    4. Please noticed that Youmi ad do NOT support Simulator, You MUST test on real mobile Device; Besides, Android pad is not appliable here for test, please test Android ad on phone only.

    1. Basic Configuration (Important)¶

    1) Import SDK¶

    Unzip the sdk, import the jar file inside libs file folder into libs of specific project.

    2) Permission Configuration¶

    Please copy the following permission configuration to AndroidManifest.xml file:

    <uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.READ_PHONE_STATE" />
<uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />
<uses-permission android:name="android.permission.ACCESS_WIFI_STATE" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.SYSTEM_ALERT_WINDOW" />

<!-- if using OfferWall, the next permission is required -->
<uses-permission android:name="android.permission.GET_TASKS" />
<uses-permission
            android:name="android.permission.PACKAGE_USAGE_STATS"
            tools:ignore="ProtectedPermissions" />

<!-- Hereunder are optional permission -->
<uses-permission android:name="com.android.launcher.permission.INSTALL_SHORTCUT" />
<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />

    3) Initialize APP info¶

    Please MUST invoke following coding in onCreate of the first Activity (First classification when starts)

    import net.youmi.android.AdManager;
...
AdManager.getInstance(Context context).init(String appId, String appSecret, boolean isTestModel);

    Tip

    P.S.

    • appId and appSecret shall be the APPID and KEY that generated on 「Youmi Developer Dashboard」
    • isTestModel: whether to enable test mode. true means YES, false means NO. (When you upload for Youmi Review or release to market, please set it to false); You can click here to learn more about TestModel.

    4) Proguard Configuration¶

    If you project using Proguard for packing, you MUST add following coding to your proguard file(P.S:proguard-project.txt), otherwise you may NOT be able to receive ad normally:

    -dontwarn net.youmi.android.**
-keep class net.youmi.android.** {
    *;
}

    In addition, you also need to invoke Android Proguard file in your proguard configuration file(P.S: project.properties)

    proguard.config=${sdk.dir}/tools/proguard/proguard-android.txt:proguard-project.txt

    5) Packing (Important)¶

    Danger

    P.S. when you pack the APK, if you assigned android:targetSdkVersion and targetSdkVersion>=17 in AndroidManifest.xml, you MUST assigned target-sdk above 17 in project.properties. Check why using target-sdk 17 for packing

    6) Setting Channel Number (Optional)¶

    Currently, Youmi has already cooperated with a lot of application store. When the target application store you want to publish in request you to add Youmi channel number, you can operate as below to package the channel number, which will make sure you can pass audit from this application store.

    Channel number list, please refer to Youmi Channel List

    Add below code in AndroidManifest.xml :

    <meta-data
    android:name="YOUMI_CHANNEL"
    android:value="This is the channel number" >
</meta-data>

    Tip

    If you stop by here, congrats! You already prepared the general configuration of Youmi Android SDK.
